# AREAS: 钢网架结构点云处理算法

这是一个用于钢网架结构点云数据处理、特征提取及质量评估的算法库。

## 算法介绍
    AREAS算法主要分为两个阶段：构件分离与变形提取。算法先利用几何特性，如：法向量与构件轴线的关系等，将点云切分成一个个独立的构件。而后剔除点云中的噪点，通过节点拟合确定构件交汇的中心点，同时解决了由于遮挡导致的构件断裂或重叠问题。最终获得每个构件完整的顶点列表和准确的端点坐标。
    一旦每个构件被独立分离，算法就会对每个独立构件进行沿构件长度方向的离散化切片，在每个切片位置计算出构件当前的实际中心点，并将其与理论上的直线轴线进行对比，计算控制点偏离理论轴线的距离，从而得出该位置的变形量。
    但需要注意，AREAS算法的适用范围是噪音较少的实验室环境点云，若噪音过多或扫描质量较低，会影响AREAS算法的精度和准确性。

## 核心功能
* **圆拟合算法**：包含 `CircleFitByTaubin.m`，采用 Taubin 法进行高精度的圆截面拟合。
* **几何特征计算**：支持角度判定 (`angle_judgement.m`)、端点计算 (`endpoint_cal.m`) 以及中点定位。
* **自动化求解**：利用 `ballsolver.m` 等核心脚本进行结构节点的空间位置还原。
